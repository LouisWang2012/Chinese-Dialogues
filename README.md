# Chinese Dialogues 对话数据集

用于训练对话模型的中文多领域对话数据集，涵盖日常生活、科技、教育等领域。
## 数据集概述
### 主要特征
✅ **多主题覆盖**：包含14大类500+主题  
✅ **真实场景**：涵盖从家常对话到专业技术的交流场景  
✅ **持续进化**：持续新增优质对话对  
✅ 双字段结构化设计  
✅ 数据集多而不大 

**应用场景**
🤖 对话系统训练
